Autovessel Web

Загружаете один XLSX, получаете ZIP с отдельными DOCX

Веб-версия работает прямо в браузере и подходит для GitHub Pages. Никакой сервер не нужен: файл обрабатывается локально у пользователя.

Как это работает

  • Сайт читает только первый лист в XLSX.
  • Находит колонки даты, судна и агентирующей компании автоматически.
  • На выходе скачивает один ZIP, внутри которого лежат отдельные DOCX.
